Simple one page TODO list created with using **Flask** and **React**

Also using Flask-SQLAlchemy extension

Instruments:
* Python == 3.7.0
* Flask-SQLAlchemy == 2.3.2
* Flask == 1.0.2
* React == 16.6.0
* Bootstrap == 4.1.3

Firstly, before you will run Flask app, run create_db.py to create main.db file.
